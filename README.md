# Event Ticketing Platform

A Laravel-based event ticketing platform where organizers can create events and users can purchase tickets.

## Planned Features

- User authentication
- Organizer dashboard
- Event creation and management
- Event listings
- Ticket purchases
- User ticket history
- Admin panel

## Tech Stack

- Laravel
- PHP
- MySQL
- Blade
- Tailwind CSS
- Vite

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve
```
